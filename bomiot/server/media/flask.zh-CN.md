# Flask

### 创建项目：

```shell
bomiot project <project_name>
```

###  项目结构

```
my-project/                    # 项目目录
├── flask_app/                 # flask app
│   └── main.py                # 主文件
...
```

- 然后你就可以直接在main.py中编写你的flask应用
